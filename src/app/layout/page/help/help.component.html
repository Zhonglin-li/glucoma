DCDB HELP


1.The homepage contains eight pull-down menus: home, search, prediction, compounds, target, pathway, help and contact in the toolbar, and users can browse relevant entries by clicking their sub-menus. The corresponding entries will be displayed.



2.In the search page, users can search various keywords, such as drug name and protein name. Alternatively, users can search a specific structure by sketching it in the JME applet window. Users can specify structure types for query, such as substructure search and similarity search, and click the “Submit” button to deliver a query molecule to DCDB Server.



3.In the prediction page, users can selcet a target and then enter a molecule in SMILES format, and finally click "Submit" buttton. Users will get comprehensive prediction results by using eight models.


4.DCDB was only test on the latest version of browser, so please try the latest browser (google chrome , firefox, etc) if you have any display problems.
